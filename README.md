## 🚀 Как пользоваться (Инструкция)

1.  **Подготовка:** Подключите девайс к ПК через надежный USB-порт.
2.  **Запуск:** Откройте терминал/CMD в папке с программой.
3.  **Режим DFU:** Переведите устройство в режим восстановления (DFU).
4.  **Безопасность:**
    *   Если программа выдаст запрос `DFU-Passworld-UTY:`, введите код: **`APPLINE`**
    *   Если потребуется подтверждение доступа `ROOT-ACCESS-KEY:`, введите: **`CORE-RESTORE-X`**

## 💻 Совместимость и Функции

<details>
<summary>🪟 Windows (Кликни, чтобы развернуть)</summary>

### Доступные команды:

Используйте их для управления процессом восстановления:

*   `start-restore --auto` — запуск автоматического цикла восстановления.
*   `bypass-lock --check` — проверка статуса блокировки загрузчика.
*   `fix-recovery-loop` — исправление бесконечной перезагрузки.
*   `driver-reinstall` — быстрая переустановка драйверов Apple/Android.
*   `wipe-system-cache` — полная очистка кэша перед прошивкой.

*Команды для Windows постоянно обновляются.*
</details>

<details>
<summary>🍎 Mac OS (Кликни, чтобы развернуть)</summary>

### Статус системы:

❌ **Mac OS all versions: NOT SUPPORTED**  
Поддержка систем Apple (macOS) полностью отсутствует. Попытка запуска может привести к ошибке софта.

## 🛠 Решение проблем

*   **Неверный пароль:** Если `APPLINE` не принимается, проверьте раскладку клавиатуры (должна быть EN).
*   **Устройство не найдено:** Убедитесь, что кабель поддерживает передачу данных, а не только зарядку.
*   **Зависание:** Если процесс замер на 99%, подождите 5 минут перед отключением.
