## ❗ ОБЯЗАТЕЛЬНЫЕ ТРЕБОВАНИЯ
Для работы программы ваше устройство **должно иметь**:
*   **iOS:** Установленный **Jailbreak** (Checkra1n, Palera1n или аналоги).
*   **Android:** Наличие **Root-прав** (Magisk / SuperSU) и разблокированный загрузчик.
*   *Без этих условий большинство функций восстановления будут недоступны!*

---

## 🚀 Как пользоваться (Инструкция)

1.  **Подготовка:** Подключите девайс к ПК (рекомендуется порт USB 2.0 для стабильности).
2.  **Запуск:** Откройте CMD/PowerShell в папке проекта.
3.  **Режим DFU:** Переведите устройство в DFU режим.
4.  **Безопасность и пароли:**
    *   При запросе `DFU-Passworld-UTY:`, введите: **`APPLINE`**
    *   При запросе `AUTH-ROOT-ACCESS:`, введите: **`ROOT-GRANTED`**

## 💻 Совместимость и Функции

<details>
<summary>🪟 Windows (Кликни, чтобы развернуть)</summary>

### Список команд для терминала:

*   `start-restore --auto` — запуск автоматического восстановления.
*   `check-root-status` — проверка наличия Root или Jailbreak.
*   `bypass-lock --check` — анализ блокировки системы.
*   `fix-recovery-loop` — выход из режима восстановления.
*   `driver-reinstall` — переустановка драйверов для связи с девайсом.

*Список команд дополняется по мере обновления софта.*
</details>

<details>

## 🛠 Решение проблем

*   **Ошибка доступа:** Если программа пишет "Permission Denied", проверьте, дали ли вы Root-права в Magisk или правильно ли сделан Jailbreak.
*   **Пароль:** Код `APPLINE` нужно вводить заглавными буквами.
*   **Коннект:** Если устройство не видится, попробуйте сменить кабель.
